Backup Tool

Faça backups de seus arquivos

Uma ferramenta escrita em Shell Script que lhe permite fazer backups de seus arquivos usando um simples comando no terminal.

Características

  • Cria uma cópia da sua pasta "/home".
  • Sistema de versionamento, que permite alterar entre versões de backups feitos em dias diferentes.
  • Permite escolher o dispositivo para armazenar as cópias.
  • Fácil uso, bastando apenas executar um arquivo para que ele inicie o backup.
  • Utiliza o sistema de empacotamento e de compressão de arquivos (tar.xz), economizando espaço em disco.
  • Feito para sistemas GNU Linux x86_64
